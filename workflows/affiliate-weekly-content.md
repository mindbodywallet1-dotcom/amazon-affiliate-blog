# Affiliate Weekly Content Workflow

## Trigger
"Run affiliate weekly content"

## Instructions

1. Read `seo/affiliate-keywords.yaml` — pick 1-2 queued articles
2. For each article:
   - Run `affiliate-product-research` if ASINs missing
   - Run `affiliate-landing-page` workflow
   - Run `affiliate-publish-checklist` if ready to go live
3. Update `kpi/affiliate-weekly-template.csv` with the week's plan
4. Rotate niches — stay multi-niche, do not cluster-lock on one category

## Weekly output target

- 1 new published page (minimum)
- 1 draft queued for next week (optional)
- Keyword cluster status updated

## Content mix rotation

| Week type | Content |
|-----------|---------|
| A | Roundup (pillar or spoke) |
| B | Single review |
| C | Comparison |
| D | Roundup in different niche |
