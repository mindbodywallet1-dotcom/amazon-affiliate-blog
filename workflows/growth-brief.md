# Monday Growth Brief

## Trigger
Run every Monday morning, or on demand: "Run growth brief"

## Instructions
1. Read `kpi/weekly-template.csv` — use the most recent row with data
2. Read `campaigns/active.yaml` and `products/catalog.yaml`
3. Write a brief to `content/week-XX/growth-brief.md`

## Output sections
1. **Last week snapshot** — sessions, revenue, CR, AOV, ad ROAS (if data exists; if empty, note "no data yet — fill kpi/weekly-template.csv")
2. **Top 3 priorities this week** — specific, actionable
3. **Content to double down on** — which posts/products to push
4. **Ad test ideas** — 2 creative angles + 1 audience hypothesis
5. **Email idea** — one broadcast topic
6. **One thing to fix** — store, offer, or funnel gap

Keep it under 400 words. No fluff.
