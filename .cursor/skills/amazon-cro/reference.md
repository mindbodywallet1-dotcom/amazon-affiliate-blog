# CRO Audit Checklist

Run before every publish:

## Above the fold
- [ ] H1 visible without scrolling on mobile
- [ ] Disclosure banner present
- [ ] Primary CTA visible without scrolling on mobile
- [ ] Hook answers search query in < 15 words

## CTAs
- [ ] At least 2 CTAs on pages over 800 words
- [ ] CTA text is descriptive (includes product name or action)
- [ ] All CTAs open in new tab (`target="_blank"`)
- [ ] Sticky CTA enabled on mobile

## Scannability
- [ ] Bulleted pros/cons for each product
- [ ] Comparison table on roundup (3+ products) and comparison pages
- [ ] H2 breaks every 200-300 words
- [ ] Quick pick highlighted on roundup pages

## Trust
- [ ] Cons section present (not pros-only)
- [ ] Last updated date shown
- [ ] No fake urgency ("only 2 left!")
- [ ] No guaranteed price claims

## Speed
- [ ] No large unoptimized images
- [ ] Page builds as static HTML (Astro default)
- [ ] Minimal third-party scripts

## Post-publish
- [ ] Log baseline sessions after 7 days
- [ ] Compare CTR before/after any CRO change
